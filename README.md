# odin-recipes
Odin Recipes
A basic recipe website built as part of The Odin Project’s Foundations curriculum. This project focuses on the core building blocks of web development: HTML structure, file navigation, and version control.

Project Description
The Odin Recipes project is a multi-page website that organizes a collection of personal recipes. It features a central homepage acting as a directory, leading to individual recipe pages that include:

High-quality images of the dishes.

Brief descriptions of the recipes.

Structured lists for ingredients (unordered) and preparation steps (ordered).

Skills Developed
Through the four iterations of this project, I have strengthened my proficiency in:

HTML Boilerplate & Structure: Utilizing semantic tags like <h1>, <h2>, <p>, <ul>, and <ol>.

File Organization: Creating and managing nested directories (e.g., the recipes/ folder) to keep project files clean.

Navigation: Implementing both internal relative links (navigating between index.html and recipe files) and external links.

Git Workflow: mastering the add, commit, and push cycle, as well as managing SSH authentication for secure repository access.

Challenges & Bug Fixes
1. SSH Authentication & Git Push
The Bug: I encountered a "Permission Denied" error when trying to clone and push my work to GitHub.
The Solution: I utilized ssh-add to re-authenticate my private key with the SSH agent, allowing for secure communication between my Linux VirtualBox and GitHub.

Technical Note: Image Links
For this specific submission, I have utilized absolute paths for the image links to ensure consistent rendering across different viewing environments.
